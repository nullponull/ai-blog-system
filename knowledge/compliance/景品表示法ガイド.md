# 景品表示法ガイド
> 最終更新: 2026-02-13
> 情報ソース: 景品表示法（不当景品類及び不当表示防止法）、消費者庁ガイドライン、2024年10月施行改正法

## 概要
景品表示法は、商品・サービスの品質や取引条件について、消費者に誤認を与える不当な表示を禁止する法律。
AI副業記事では「収入額の誇大表示」「体験談の不適切な使用」「打消し表示の不備」が主なリスクとなる。
2024年10月の改正で直罰規定（100万円以下の罰金）が新設され、違反リスクが大幅に増大している。

## 記事生成時の適用ルール
- 具体的な収入額を記載する場合、必ず根拠データと条件を明示すること
- 「誰でも」「簡単に」「確実に」等の断定的表現は使用禁止
- 体験談を使用する場合、「個人の感想であり、効果を保証するものではありません」等の打消し表示を必ず付記すること
- 打消し表示は本文と同等以上のサイズ・視認性で表示すること（8ポイント以上）
- 「平均的な結果」が体験談と大きく異なる場合は、平均的な結果も併記すること
- 収入に関する表現は「可能性がある」「目指せる」等の表現に留めること
- 「月XX万円」等の具体額は、実証データがない限り使用しないこと

## 禁止表現・注意表現

### 優良誤認表示（品質・効果の過大表示）
- NG: 「AIで月30万円稼げる方法」 → 合理的根拠なしの収入保証は優良誤認
- NG: 「誰でも簡単に副収入が得られる」 → 個人差がある事項の断定は優良誤認
- NG: 「ChatGPTだけで不労所得を実現」 → 実態と異なる表示
- NG: 「初月から10万円達成した方法を公開」 → 再現性の保証がない体験談
- NG: 「プロ級のスキルが身につく」 → 根拠のない品質表示
- OK: 「AI副業で収入を得るための具体的な手順を解説します」 → 事実の範囲内
- OK: 「筆者の場合、3ヶ月の実践で月5万円の収入を得ました（個人の実績であり、同様の結果を保証するものではありません）」
- OK: 「AIツールを活用した副業には様々な可能性があります。ただし成果には個人差があります」

### 有利誤認表示（取引条件の過大表示）
- NG: 「今だけ無料！通常価格10万円の教材」 → 通常販売実績がない場合は二重価格の有利誤認
- NG: 「期間限定90%OFF」 → 常時セール状態の場合は有利誤認
- NG: 「先着100名限定」 → 実際には制限がない場合は有利誤認
- OK: 「この記事は無料で公開しています」 → 事実の表示
- OK: 「有料部分では具体的な手順を解説しています」 → 事実の表示

### 打消し表示の要件
- 本文の強調表示に近接した位置に配置すること
- 本文と同等のフォントサイズ（最低8ポイント以上）で表示すること
- 背景色と十分なコントラストがあること
- 内容が強調表示と矛盾しないこと
- スマートフォン表示でも視認可能なサイズであること

### 体験談使用のルール（2023年消費者庁改正ガイドライン対応）
- 体験談は「個人の感想です」だけでは不十分
- 体験談と「大多数の消費者が得られる結果」が異なる場合は、平均的な結果も明示が必要
- 「個人差があります」「結果を保証するものではありません」に加え、典型的な結果を併記すること
- 体験者が実在し、体験内容が事実であることが前提

## 詳細情報

### 法的根拠
- **景品表示法第5条第1号**: 優良誤認表示の禁止
- **景品表示法第5条第2号**: 有利誤認表示の禁止
- **景品表示法第5条第3号**: 指定告示による禁止（ステマ規制含む）
- **不実証広告規制（第7条第2項）**: 表示の裏付けとなる合理的根拠資料の提出義務

### 2024年10月改正の主要ポイント
1. **直罰規定の新設**: 優良誤認・有利誤認表示に対し100万円以下の罰金（刑事罰）
2. **確約手続の導入**: 事業者が自主的に是正措置計画を提出・認定されれば措置命令を回避可能
3. **課徴金の加重**: 10年以内の再違反で課徴金1.5倍
4. **適格消費者団体の権限強化**: 差止請求権の拡充

### 措置命令・課徴金の実態
- 措置命令: 誤認の排除、再発防止策の実施、同様の違反行為の禁止を命令
- 課徴金: 対象商品の売上額の3%（2024年8月-2025年7月の合計約3億3,348万円）
- 措置命令に従わない場合: 2年以下の懲役または300万円以下の罰金

### 行政処分事例

#### 事例1: 情報商材の収入誇大表示
副業系情報商材において「月収100万円可能」等の表示について、消費者庁が不実証広告規制に基づき合理的根拠の提出を求め、十分な根拠が示されなかったとして措置命令が発出された事例が複数存在。

#### 事例2: アフィリエイト広告の優良誤認
2024年、消費者庁はアフィリエイト広告を通じた優良誤認表示について、広告主に対して措置命令を発出。広告主がアフィリエイターの広告内容を把握していなかった場合でも、広告主が責任を負うことが確認された。

#### 事例3: 大正製薬ステマ措置命令（2024年11月）
大正製薬株式会社に対し、消費者庁がステルスマーケティング規制違反で再発防止の措置命令を発出。

### AI副業記事で特に注意すべきパターン
1. **スキル習得期間の過小表示**: 「たった1週間で」等の表現は、多くの場合実態と乖離
2. **必要投資額の過小表示**: ツール費用、学習時間等のコストを隠す
3. **成功率の過大表示**: 全体の平均ではなく成功例のみを取り上げる
4. **AIツールの能力の過大表示**: AIが自動的に全てやってくれるかのような表現
5. **市場飽和リスクの隠蔽**: 参入者増加による収益低下の可能性を記載しない
