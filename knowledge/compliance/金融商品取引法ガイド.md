# 金融商品取引法ガイド
> 最終更新: 2026-02-13
> 情報ソース: 金融商品取引法、金融庁「金融商品取引業者等向けの総合的な監督指針」、財務省関東財務局Q&A

## 概要
金融商品取引法（金商法）は、投資助言・代理業を含む金融商品取引業を規制する法律。
副業記事で投資関連サービス（DMM株、SBI証券等）に言及する場合、「投資助言」に該当しない表現を使用する必要がある。
無登録で投資助言業を行った場合、5年以下の懲役または500万円以下の罰金という重い罰則がある。

## 記事生成時の適用ルール
- 特定銘柄の売買推奨は絶対に行わないこと
- 「この株を買えば儲かる」等の投資判断を示す表現は使用禁止
- 投資関連サービスに言及する場合は、一般的な情報提供の範囲に留めること
- 投資のリスクについて、利益と同等の視認性で記載すること
- 「投資は自己責任です」等の免責表現を必ず付記すること
- 有料記事で投資手法を解説する場合は、投資助言業に該当しないか慎重に判断すること
- 投資成績の表示は実際のデータに基づき、再現性を保証する表現は避けること

## 禁止表現・注意表現

### 投資助言に該当するおそれのある表現（NG）
- NG: 「今すぐA社の株を買うべき」 → 特定銘柄の売買推奨は投資助言に該当
- NG: 「このポートフォリオで年利20%が狙えます」 → 具体的な投資判断の提供
- NG: 「会員限定: 今週の推奨銘柄リスト」 → 有料・会員制での銘柄推奨は投資助言業
- NG: 「AIが分析した結果、この銘柄は上がります」 → 投資判断の提供
- NG: 「DMM株で口座を開設して、この銘柄を買ってください」 → 具体的な投資指示
- NG: 「FXで月10万円稼ぐ確実な方法」 → 確実性を示す収益保証
- NG: 「ビットコインは必ず値上がりする」 → 価格予測の断定

### 投資助言に該当しない表現（OK）
- OK: 「投資を始める際に検討すべきポイントを解説します」 → 一般的な情報提供
- OK: 「証券口座の開設手順と各社の特徴を比較します」 → サービスの客観的比較
- OK: 「分散投資の考え方について基本的な知識を紹介します」 → 投資教育の範囲
- OK: 「筆者はインデックス投資を行っていますが、投資は各自の判断でお願いします」 → 体験共有+免責
- OK: 「NISAの制度概要と始め方について解説します」 → 制度の説明
- OK: 「投資にはリスクが伴います。元本割れの可能性があることをご理解ください」 → リスク表示

### 免責表現の推奨パターン
- OK: 「本記事は情報提供を目的としており、特定の金融商品の売買を推奨するものではありません」
- OK: 「投資判断はご自身の責任において行ってください」
- OK: 「本記事の情報は執筆時点のものであり、最新の情報は各サービスの公式サイトでご確認ください」
- OK: 「金融商品には元本割れのリスクがあります。投資は余裕資金で行い、リスクを十分にご理解の上でご判断ください」

## 詳細情報

### 投資助言業の定義と該当性判断

#### 金融商品取引法上の定義
投資助言・代理業とは、金融商品の価値等の分析に基づく投資判断に関し、口頭、文書その他の方法により助言を行うことを約し、相手方がそれに対し報酬を支払うことを約する契約を締結し、当該契約に基づき助言を行う業務。

#### 投資助言業に該当する要件
以下の全てを満たす場合、投資助言業に該当する可能性が高い:
1. **有価証券の価値等又は金融商品の価値等の分析に基づく投資判断**を提供
2. **報酬**を受け取る（金銭に限らず、商品提供等も含む）
3. **契約**に基づく（明示的な契約がなくても実質的に契約関係にある場合を含む）
4. **業として行う**（反復継続して行う）

#### 投資助言業に該当しないケース（金融庁監督指針）
金融庁の監督指針VII-3-1において、以下は投資助言業の登録を要しないとされている:

1. **不特定多数向けの情報提供**: 不特定多数の者を対象として、不特定多数の者が随時に購入可能な方法により提供される投資情報
2. **店頭販売の書籍・雑誌**: 誰でも購入可能な書籍等での投資情報
3. **無料のブログ・メルマガ**: 投資顧問料を受け取らない無料の情報提供
4. **ダウンロード可能なソフトウェア**: 不特定多数が購入可能なツール

#### 判断が微妙なケース
- **有料のnote記事で投資手法を解説**: 不特定多数が購入可能であれば投資助言に該当しない可能性があるが、個別の投資判断を含む場合は注意が必要
- **会員制サロンでの投資情報提供**: 会員登録が必要な場合、「不特定多数が随時に購入可能」の要件を満たさない可能性があり、投資助言業に該当するリスクが高い
- **有料メルマガでの銘柄推奨**: 個別の投資判断を提供し報酬を得ている場合は投資助言業に該当する可能性

### DMM株などの投資関連サービスに言及する際の注意点

#### アフィリエイト広告の場合
- 証券口座開設のアフィリエイトは、サービスの紹介であり投資助言そのものではない
- ただし、「この口座で○○株を買えば儲かる」等の表現は投資助言に該当
- サービスの機能比較、手数料比較等の客観的情報は問題なし
- ステマ規制のPR表記は必須（別途ステマ規制ガイド参照）

#### 推奨される言及方法
- サービスの特徴・手数料・使い勝手等の客観的比較
- 「筆者が利用している理由」という体験共有（投資判断ではなくサービス選択の理由）
- 制度（NISA、iDeCo等）の説明と対応サービスの紹介
- 必ずリスク表示と免責表現を付記

### 「投資は自己責任」等の免責表現のベストプラクティス

#### 効果的な免責表現の要件
1. **視認性**: 記事内で目立つ位置（冒頭または末尾）に配置
2. **具体性**: 「自己責任」だけでなく、具体的なリスクを記載
3. **網羅性**: 元本割れリスク、価格変動リスク、流動性リスク等を網羅
4. **誠実性**: 形式的な免責ではなく、読者の利益を考慮した表現

#### 推奨される免責表現テンプレート
```
【投資に関する注意事項】
本記事は情報提供を目的としたものであり、特定の金融商品の売買を推奨するものではありません。
金融商品への投資には、元本割れを含むリスクが伴います。
投資判断はご自身の責任において、十分な調査と検討の上で行ってください。
本記事の情報は執筆時点のものであり、正確性・完全性を保証するものではありません。
```

### 無登録で投資助言業を行うリスク

#### 罰則
- **個人**: 5年以下の懲役もしくは500万円以下の罰金、またはその併科
- **法人**: 5億円以下の罰金
- 金融商品取引法第197条の2に基づく

#### 行政処分事例
- 無登録で投資助言を行っていた個人・法人に対し、関東財務局が警告を発出した事例が複数存在
- SNSやオンラインサロンを通じた投資助言が増加しており、規制当局の監視が強化されている
- 「投資助言ではなく教育」と主張しても、実質的に投資判断を提供している場合は該当する

#### AI副業記事での特有のリスク
- 「AIで投資判断を自動化する副業」等のテーマは、投資助言業との境界が曖昧
- AIトレーディングツールの紹介は、ツールの機能説明に留め、具体的な投資判断の提供は避ける
- 「AIを使えば投資で稼げる」等の表現は、景品表示法と金商法の両面でリスクが高い
