# 信頼できるソース一覧
> 最終更新: 2026-02-13
> 情報ソース: Web検索による実在ソースの確認済みリスト

## 概要
AI副業系記事の作成・ファクトチェックに使用する信頼できる情報ソースの一覧。
公的機関、調査会社、メディア、プラットフォーム公式データ、海外ソースに分類し、
各ソースの信頼度・更新頻度・利用可能なデータ種類を明記する。

## 記事生成時の活用方法
1. 数値データを記事に含める際は、必ずこのリストの「信頼度: 高」のソースから引用する
2. 市場規模・統計データは複数ソースでクロスチェックする
3. ソース引用時は「〇〇によると」と出典を明記する
4. 海外データは為替レート・市場環境の違いに注意して引用する
5. 更新頻度を確認し、古いデータを最新と偽らない

## 6. ソース利用時の注意事項

### 信頼度の判断基準

| 信頼度 | 基準 |
|--------|------|
| 高 | 政府機関・上場企業・大手調査会社の公式データ。一次情報。 |
| 中-高 | 業界団体・専門メディアの調査。方法論が明確なもの。 |
| 中 | 個別企業のプレスリリース・ブログ。バイアスの可能性あり。 |
| 低 | 個人ブログ・SNS投稿。裏付けなしの数値。記事生成には使用しない。 |

### データ引用のルール

1. **必ず出典を明記**: 「〇〇（2025年）によると」の形式で引用
2. **データの鮮度を確認**: 2年以上前のデータは「〇〇年時点」と明記
3. **予測値と実績値を区別**: 「予測では」「実績として」を使い分け
4. **複数ソースでクロスチェック**: 重要な数値は2つ以上のソースで確認
5. **海外データの換算注意**: 為替レート・市場定義の違いに注意
6. **調査の対象・方法論を確認**: サンプル数、調査方法を把握してから引用

### 使用禁止パターン

- 出典不明の「〇〇万円稼げる」系の数値
- 個人の成功体験を一般化した統計的主張
- AI副業の収入を保証するような表現
- 古いデータを最新かのように引用すること
- 海外データをそのまま日本市場に適用すること
